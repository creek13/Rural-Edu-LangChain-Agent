# Rural-Edu-LangChain-Agent
# 🌱 Rural Learning Agent（山村自学助手）

一个基于 AI 的学习助手，旨在帮助**教育资源匮乏地区的学生进行自学**。

---

## 📌 项目背景

发起人在2025年暑假曾跟随校内支教团在甘肃双岘镇支教，发现：在偏远山村地区，学生面临：

* 缺乏优质教师指导（年轻教师往往呆不到两年就会离开）
* 学习反馈不足（家庭与学校因家长文化水平、老师的稀缺无法提供有效的反馈）
* 遇到问题容易放弃
* 基础知识断层严重

本项目尝试通过 AI，提供一种低成本、可扩展的解决方案：

> **让学生在没有老师的情况下，也能完成基础学习闭环。**

---

## 🎯 项目目标

这个 Agent 的目标不是简单答题，而是：

* 帮助学生 **理解知识**
* 引导学生 **自己思考**
* 提供 **持续反馈**
* 降低学习门槛，提高坚持概率

核心理念：

> ❌ 不是“告诉答案”
> ✅ 而是“帮助学生自己做出来”

---

## 🧠 核心设计原则

1. **分步骤引导**

   * 不直接给答案
   * 将问题拆解为可理解的小步骤

2. **低门槛解释**

   * 使用生活化语言
   * 避免复杂术语

3. **基础优先**

   * 自动识别知识断层
   * 补充前置知识

4. **正反馈机制**

   * 鼓励尝试
   * 降低挫败感

---

## ⚙️ 当前实现（MVP）

当前版本基于 Dify 构建 Chat App，具备：

* 基础问答能力
* 分步骤教学引导
* 简化表达与结构化输出
* 针对弱基础用户的优化提示词

---

## 🚀 使用方式

1. 打开 Chat App
2. 输入学习问题（如数学题、概念疑问）
3. 跟随引导逐步思考
4. 尝试自己完成问题

建议：

* 不要只看答案
* 尝试回答 Agent 提出的问题

## 🤔 为什么做这个项目？

这个项目的出发点很简单：

> **既然AI能帮助像我一样的大学生学习，那它就有潜力胜任中小学阶段的老师**

如果一个学生：

* 有时间
* 有努力
* 但没有资源

那么 AI 应该成为那个“最基础的老师”。
